dependencies:

--> express: Framework Node.JS que fornece vários recursos pra desenvolvimento de aplicações web com Node;

--> body-parser: É um Middleware (uma "camada no meio" que facilita a comunicação entre duas aplicações,

--> fs: File System é um módulo pra se trabalhar com arquivos. Com ele é possível criar, ler, atualizar, 
renomear e deletar arquivos em nosso computador;

--> ejs: pacote responsável pela engine EJS
